An dynamic and informative business website with a product and job vacancy database, built using Next.js (React), Prisma, Emotion JS, MySQL, and Bootstrap..


It's a project built in Next.js (React) for a butcher shop. They have different branches, and the idea is to display all of them from a single page. The website is still under construction, currently at 60% completion. It features a database that operates dynamically to manage products and job vacancies. This database is managed using an ORM (Prisma) to make its handling easier and integrate the back-end with the front-end. The styles were made using Bootstrap and Emotion to create styled components.
