[Github Repo](https://github.com/Christian1209/ParkingLot_Control)

A desktop application built with Windows Forms (C#) using the .NET framework, it's a system for managing parking spaces, including spots for disabled individuals, preferred spaces, vehicle registration, and total sales. It doesn't have a database because it was created for academic purposes

## Payment Management

![HomePage](https://res.cloudinary.com/dudl1fetz/image/upload/f_auto,q_auto/v1/Parking/nzlrubxfcx0coss9rruu)
"Here, charges are made according to the ticket, and we have information about the ticket, time, cost, type of vehicle, etc. Additionally, a discount can be applied if a ticket is presented."

## Ticket Generation

![HomePage](https://res.cloudinary.com/dudl1fetz/image/upload/f_auto,q_auto/v1/Parking/me5298f3lcc8spn5kikn)
A ticket is generated with the vehicle's characteristics, indicating if the customer is a frequent visitor, elderly, disabled, or if they are heading to the pension area.


## Real-Time Map of parking availability.

![HomePage](https://res.cloudinary.com/dudl1fetz/image/upload/f_auto,q_auto/v1/Parking/u5agv18q2lqwtwjuedju)
Here you can see the available spaces. This map is dynamically updated as tickets are processed and will be freed up as they are paid
