"A desktop application developed in Windows Forms (C#) with the .NET framework, for managing a store, enables administrative control of the business, a registry of customers and suppliers, and inventory. It is connected to a MySQL database utilizing .NET drivers to facilitate the connection. It was created for academic purposes.
